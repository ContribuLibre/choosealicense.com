C’est quoi, au juste ?

GitHub souhaite aider les développeurs à choisir une licence open source pour leur code source.

Si vous savez déjà ce que vous faites et avez une licence que vous préférez utiliser, c’est parfait ! Nous ne sommes pas là pour vous faire changer d’avis. Mais si le grand nombre de choix vous déconcerte, nous pouvons peut-être vous aider.

Liste non exhaustive

Ce site n’est pas un répertoire exhaustif des licences open source. La grande majorité des projets seront probablement bien servis par l’une des trois options mises en avant sur la page d’accueil — choisir la licence privilégiée par des projets similaires au vôtre, ou la licence permissive la plus populaire (MIT), ou la licence copyleft la plus populaire (GNU GPLv3). Au cas où vous auriez des besoins spécifiques non couverts par celles-ci, nous mettons aussi en avant quelques autres licences à envisager et proposons une page sur les licences pour les projets non logiciels.

Consultez notre annexe pour un tableau de toutes les licences cataloguées dans le dépôt choosealicense.com, ainsi que les liens ci-dessous pour encore plus de licences que vous n’avez pas besoin de connaître pour autant choisir une excellente licence pour votre projet.

Aidez-nous à l’améliorer

Choosealicense.com ne parle pas seulement d’open source : le site lui-même est open source. Vous voyez quelque chose qui pourrait être mieux fait ? N’hésitez pas à forker le projet sur GitHub et à proposer une pull request. Vos améliorations sont les bienvenues.

Avertissement

Nous ne sommes pas juristes. Enfin, la plupart d’entre nous. L’objectif de ce site n’est pas de fournir des conseils juridiques. Son but est d’offrir un point de départ pour vous aider à faire un choix éclairé, en présentant de façon simplifiée des informations sur les licences open source populaires. Si vous avez des questions sur la meilleure licence pour votre code, ou sur toute autre question juridique le concernant, il vous appartient de poursuivre vos recherches ou de consulter un professionnel.
