Légende

Les licences open source accordent au public des ✓ permissions de faire, avec les œuvres sous licence, des choses que le droit d’auteur ou d’autres lois de « propriété intellectuelle » pourraient sinon interdire.

L’octroi des permissions par la plupart des licences open source est subordonné au respect de ⓘ conditions.

La plupart des licences open source comportent aussi des ✕ limitations qui, généralement, excluent toute garantie et toute responsabilité, et parfois excluent expressément les brevets ou les marques de l’octroi de la licence.

✓ Le contenu sous licence peut être distribué.
Modification
✓ Le contenu sous licence peut être modifié.
Usage des brevets
✓ Cette licence accorde expressément des droits de brevet de la part des contributeurs.
Usage privé
✓ Le contenu sous licence peut être utilisé et modifié de façon privée.

Conditions

Fournir le code source
ⓘ Le code source doit être rendu disponible lors de la distribution du contenu sous licence.
Citer auteur et licence
ⓘ Une copie de la licence et de la mention de droit d’auteur doit être incluse avec le contenu sous licence.
Citer auteur et licence (source)
ⓘ Une copie de la licence et de la mention de droit d’auteur doit être incluse avec le contenu sous licence sous forme de code source, mais n’est pas requise pour les binaires.
L’usage réseau vaut distribution
ⓘ Les utilisateurs qui interagissent avec le contenu sous licence via un réseau ont le droit d’en recevoir le code source.
Même licence
ⓘ Les modifications doivent être publiées sous la même licence lors de la distribution du contenu sous licence. Dans certains cas, une licence similaire ou apparentée peut être utilisée.
Même licence (fichier)
ⓘ Les modifications des fichiers existants doivent être publiées sous la même licence lors de la distribution du contenu sous licence. Dans certains cas, une licence similaire ou apparentée peut être utilisée.
Même licence (bibliothèque)
ⓘ Les modifications doivent être publiées sous la même licence lors de la distribution du contenu sous licence. Dans certains cas, une licence similaire ou apparentée peut être utilisée, ou cette condition peut ne pas s’appliquer aux œuvres qui utilisent le contenu sous licence comme bibliothèque.
Indiquer les modifications
ⓘ Les modifications apportées au contenu sous licence doivent être documentées.
