这是关于什么的？

GitHub 希望帮助开发者为他们的源代码选择一个开源许可证。

如果你已经心中有数，并且有偏好使用的许可证，那很好！我们无意改变你的想法。但如果你被众多的选择弄得不知所措，也许我们能帮上忙。

并不全面

本站并非开源许可证的完整目录。绝大多数项目很可能都能由首页上重点推荐的三个选项很好地满足——选择与你类似的项目所偏好的许可证、最受欢迎的宽松许可证（MIT）、或最受欢迎的 copyleft 许可证（GNU GPLv3）。万一你有这些选项未能覆盖的特定需求，我们也重点介绍了另外几个值得考虑的许可证，并设有一个关于非软件项目许可证的页面。

请查阅我们的附录，那里有一张表格收录了 choosealicense.com 仓库中编目的所有许可证；并参阅下方的链接，那里有更多许可证——你无需了解它们也能为你的项目选到一个很好的许可证。

帮助我们改进它

Choosealicense.com 不仅仅是在谈论开源；本站本身也是开源的。看到了你觉得可以做得更好的地方？欢迎在 GitHub 上复刻本项目并提交 pull request。我们欢迎你的改进。
