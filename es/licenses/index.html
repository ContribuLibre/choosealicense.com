Licencias

Las licencias de código abierto conceden permiso a cualquiera para usar, modificar y compartir el software licenciado con cualquier fin, sujeto a condiciones que preservan la procedencia y la apertura del software. Las siguientes licencias se ordenan por número de condiciones, de la que más tiene (GNU AGPLv3) a ninguna (Unlicense). Observa que las licencias populares destacadas en la página de inicio (GNU GPLv3 y MIT) se sitúan dentro de este espectro.

Si buscas una tabla de referencia con todas las licencias de choosealicense.com, consulta el apéndice.

GNU AGPLv3

Los permisos de esta licencia copyleft, la más fuerte, están condicionados a poner a disposición el código fuente completo de las obras licenciadas y de sus modificaciones —incluidas las obras más amplias que usen una obra licenciada— bajo la misma licencia. Los avisos de copyright y de licencia deben conservarse. Quienes contribuyen conceden expresamente derechos de patente. Cuando se usa una versión modificada para prestar un servicio a través de una red, debe ponerse a disposición el código fuente completo de esa versión modificada.
