Sugerir esta licencia

Abre una pull request para sugerir esta licencia a un proyecto que no tenga licencia. Sé cortés: comprueba si ya se ha sugerido una licencia, intenta proponer una adecuada para la comunidad del proyecto y mantén una comunicación amable con los mantenedores.

Cómo aplicar esta licencia

Insert the license or a link to it along with a copyright notice into your source file(s), and/or create a text file (typically named LICENSE or LICENSE.txt) in the root of your source code and copy the text of the license into the file, replacing [year] with the current year and [fullname] with the name (or names) of the copyright holders.

Pasos opcionales

It is recommended to add a link to the license and copyright notice at the top of each source file, example text can be found at https://oss.oracle.com/licenses/upl/.

Añade UPL-1.0 a la descripción del paquete de tu proyecto, si procede (por ejemplo, Node.js, Ruby y Rust). Así la licencia se mostrará en los directorios de paquetes.
