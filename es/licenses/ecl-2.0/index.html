Sugerir esta licencia

Abre una pull request para sugerir esta licencia a un proyecto que no tenga licencia. Sé cortés: comprueba si ya se ha sugerido una licencia, intenta proponer una adecuada para la comunidad del proyecto y mantén una comunicación amable con los mantenedores.

Cómo aplicar esta licencia

Create a text file (typically named LICENSE or LICENSE.txt) in the root of your source code and copy the text of the license into the file.

Pasos opcionales

The Apereo Foundation recommends taking the additional step of adding a boilerplate notice to the header of each source file. You can find the notice in the appendix at the very end of the license text.

Añade ECL-2.0 a la descripción del paquete de tu proyecto, si procede (por ejemplo, Node.js, Ruby y Rust). Así la licencia se mostrará en los directorios de paquetes.
