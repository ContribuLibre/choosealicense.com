Vim License

There are no restrictions on using or distributing an unmodified copy of the software. Parts of the software may also be distributed, but the license text must always be included. For modified versions a few restrictions apply. The license is GPL compatible, you may compile the software with GPL libraries and distribute it.

Sugerir esta licencia

Abre una pull request para sugerir esta licencia a un proyecto que no tenga licencia. Sé cortés: comprueba si ya se ha sugerido una licencia, intenta proponer una adecuada para la comunidad del proyecto y mantén una comunicación amable con los mantenedores.

Cómo aplicar esta licencia

Create a text file (typically named LICENSE or LICENSE.txt) in the root of your source code and copy the text of the license into the file. Replace [project] with the project name.

Pasos opcionales

Añade Vim a la descripción del paquete de tu proyecto, si procede (por ejemplo, Node.js, Ruby y Rust). Así la licencia se mostrará en los directorios de paquetes.
