Sugerir esta licença

Abra um pull request para sugerir esta licença a um projeto que não tem licença. Seja gentil: verifique se uma licença já foi sugerida, tente sugerir uma adequada à comunidade do projeto e mantenha uma comunicação cordial com as pessoas que mantêm o projeto.
