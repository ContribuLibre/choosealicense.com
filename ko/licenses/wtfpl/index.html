이 라이선스 제안하기

라이선스가 없는 프로젝트에 이 라이선스를 제안하려면 풀 리퀘스트를 보내세요. 정중하게 진행해 주세요. 이미 라이선스가 제안되었는지 확인하고, 프로젝트의 커뮤니티에 맞는 라이선스를 제안하려고 노력하며, 프로젝트 메인테이너와 우호적으로 소통하세요.

이 라이선스를 적용하는 방법

Create a text file (typically named LICENSE or LICENSE.txt) in the root of your source code and copy the text of the license into the file.

선택 단계

해당되는 경우 프로젝트의 패키지 설명에 WTFPL을(를) 추가하세요(예: Node.js, Ruby 및 Rust). 이렇게 하면 패키지 디렉터리에 라이선스가 표시됩니다.
